DirectX 11 C++ Space Shooter

Summary

For this project I have created an application that demonstrates my skills in writing code in C++ whilst using Directx 11. I created a small scene located in space with the user represented as a spaceship with a few extra interactable objects.

Movement, rotation and scale were all implemented to allow me to explore how Matrices could be used. This was done in the form of a World View Projection Matrix.

Project Outcomes

  • I have gained a core understanding of how a World View Project Matrix can be minipulated.
  • Multiple transform matrices have been used in the form of multiple cameras.
  • Implementation of UV Mapping into a 3D scene.
  • Ambient, Directional and Diffuse lighting are used.
