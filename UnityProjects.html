Unity Projects

Mr. Blob

Mr. Blob is a game that I made for mobile use. It is a short demo that demonstrates what was implemented through this project. Some of the features include:

- Mr. Blob will always look towards your touch

- Immersive Player to Blob interactions using finger

- Use of Accelerometer to make Mr. Blob slide

- Mr. Blob simulates AI through nodes and scripts


Matrix Rotation and Translation

In this project I created a Grid-System to learn more about how a Matrix can be minipulated. The knowledge gained improves on what was learnt on my other project Animal Behaviours.

During the implementation I learnt:

- How to/not-to organise object within a grid

- How to make objects cascade down.

- How to rotate a Matrix and update its values.

3D World Navmesh

A Navmesh implementation into a 3D scene. The Navmesh is baked into the floor of an office allowing entities to move around. A state machine is then used to give the entities an idea of where to go. All 3D models, Textures and assets where created myself.

- State Machine incorperated alongside the Navmesh

- Research into and creating of 3D models in 3DS Max and Blender

- Multiple Cameras used

- Desktop and Android Mobile friendly
