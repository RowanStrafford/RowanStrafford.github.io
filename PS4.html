PlayStation 4 Development Kit Project (C++)

Summary

For this project a PS4 Dev Kit was used to create an application that could be used on the Console with a controller.

I incorperated controller vibration and colour bar changes to enhance the gameplay. Multi-Threading was implemented to allow an animation to play on the loading screen as the assets are created.

Outcomes

  • Understanding of how a Development Kit is used.
  • Multiple controller set-ups for different controller types and buttons.
  • More than just the main thread used.
