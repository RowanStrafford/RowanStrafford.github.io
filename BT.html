Behaviour Tree in C++ and DirectX 11

Summary

This project involved the designing, planning, implementation and testing of a Behaviour Tree. This concept was used to simulate animal behaviours such as walking, drinking, hunting and many more with the tree consisting of more than 15 different nodes.

DirectX 11 with C++ was used to implement the application allowing the animals to take on a visual form as entities. Many complex mathmatical functions were learnt and understood for this project including Matrix mulitplication and flocking calculations.

Outcomes

  • Two Behaviour Trees (Carnivore and Herbivore)
  • Procedurally generated map allowing entities to provide unique outcomes
  • Flocking implemented using Cohesion, Seperation and Alignment
  • Extensive research into Reinforcement Learning
  • Multiple Node classes created allowing ease in setting up new behaviours
