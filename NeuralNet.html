Neural Network (C++)

Summary

A Neural Network has been implemented which ajusts itself to calculate for a squared result. The training data simply consists of a random integer as the input and its result, the input squared, as its output.

The Neural Network successfully used the Feed Forward and Back Propagation algorithms to adjust its weights. This project has been my biggest challenge and I will continue to grow it.

Outcomes

  • New understanding of non-linear functions.
  • Extensive research into Machine Learning
  • Side program that generates the training data.
  • Visual element built into the Net.
